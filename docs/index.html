Sky: Children of the Light Companion

Isekai is a lightweight desktop companion for Sky players. Track recurring events, plan goals and collections, and keep important timers nearby in a local app built with Tauri, Rust, and React.

Built for every Sky journey

Timers, planning, collections, and quick desktop access in one focused companion.

Fast Rust core

A native Rust core keeps event calculations and desktop behavior responsive without weighing down your system.

Always-near overlay

Keep important timers visible in a passive transparent overlay with global shortcut access.

Event countdowns

Planner and goals

Collection tracking

Offline-friendly

Frequently Asked Questions

Do I need a Sky account or login?

No. Isekai does not ask for your Sky login or connect to your game account.

Does Isekai work offline?

Most features are local, including event calculations, planner data, collection progress, themes, and saved settings.

Which platforms are supported?

Isekai supports Windows, macOS, and Linux. The latest release page lists the installer formats currently available for each platform.

Why does macOS warn when I open Isekai?

Current macOS builds are unsigned. After moving Isekai to Applications, follow the one-time Gatekeeper command in the project README if macOS blocks it.

Is Isekai an official Sky app?

No. Isekai is an unofficial fan-made utility. Sky: Children of the Light belongs to thatgamecompany.

Where can I download it or get help?

Download Isekai from this page or GitHub Releases. For help, join the Discord community or message the Telegram channel linked below.
